#DevOps End-to-End CI/CD Pipeline

In this project, infrastructure was provisioned on AWS using Terraform with an Infrastructure as Code (IaC) approach. A distributed Jenkins master–agent architecture was configured using Ansible. CI/CD pipelines were implemented with Jenkinsfile to automate build, test, SonarQube code analysis, and artifact publishing to JFrog Artifactory. Applications were containerized with Docker and deployed to Kubernetes. Prometheus and Grafana were used to provide monitoring and observability.

Technologies:
Terraform, AWS, Ansible, Jenkins, JFrog Artifactory, SonarQube, Docker, Kubernetes, Prometheus, Grafana
